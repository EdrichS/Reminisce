This personal project was made because my girlfriend wanted to view all our previous text to reminisce on our first conversation and memorable moments. Usually, we can do this by just going to our messages app and scrolling back to prior conversations or searching up key words to go back to certain topics. The challenge with this is it takes time and sometimes it doesnt work. My girlfriend also deleted her texts from her messages because of family problems. As a solution and fun project, I developed this.

This code is designed to be fully on Mac.
